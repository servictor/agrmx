Adicione as imagens com o nome de arquivo correto e correspondente ao http referenciado em texto dos serviços.

Exemplo:

[
    {
      "title": "Produtos disponiveis nacionalmente",
      "subtitle": "Poupas de Frutas Congeladas",
      "image": "https://servictor.github.io/agrmx/imagens-servicos/poupas-de-frutas.png",
      "order": 1
    },
    {...}
  ]

poupas de frutas é um arquivo que vive aqui nesta pasta, e tem que ter o memo nome descrito no http.
